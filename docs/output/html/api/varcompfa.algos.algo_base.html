update(context)[source]¶

Update from new experience. Uses the supplied parameters from context which are then fed to the agent’s learn function according to _learn_params.

Via some metaclass programming, we determine the signature of learn at class creation, and define _learn_params accordingly.

Parameters:context (dict) – A dictionary containing all information needed by self.learn.
Returns:The value returned by the algorithm’s learn() method.
