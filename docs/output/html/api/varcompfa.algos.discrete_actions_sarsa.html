varcompfa.algos.discrete_actions_sarsa module¶

SARSA using LFA with discrete actions.

class varcompfa.algos.discrete_actions_sarsa.DiscreteSARSA(num_features, num_actions, epsilon=0.05)[source]¶

Bases: varcompfa.algos.algo_base.LearningAlgorithm

SARSA with linear function approximation.

Actions are assumed to be discrete, while states are represented via a feature vector. That is, Q(s,a) = [〈w, x〉]_a Exploration occurs via an ε-greedy policy.
