varcompfa.policies.discrete_actions_fixed_policy module¶

Fixed policies for environments with discrete actions.

class varcompfa.policies.discrete_actions_fixed_policy.DiscreteFixed(policy)[source]¶

Bases: object

A (possibly stochastic) policy that chooses actions via table-lookup.

Policies are specified in terms of dictionaries with states/observations as keys. The values are specified in terms of a sequence of tuples, with each tuple having the form (action, weight), such that the weights are nonnegative and proportional to the frequency that action should be selected in the given state.

Notes

Since we’re using dicts to specify the policy, it requires the keys to be hashable and to completely cover the visitable state-space.
