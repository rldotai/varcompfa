varcompfa.envs.paper_chain module¶

A simple Markov Decision Process environment.

A five-state chain which always transitions closer to the terminal state with a reward generated via N(1, 1). This makes the values and variance easy to calculate (although it’s not properly ergodic, nor is it aperiodic).
