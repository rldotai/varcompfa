class varcompfa.engine.parameters.EpisodicExponential(initial_val, decay_rate, terminal_value=None)[source]¶

Bases: object

A parameter that decays exponentially with the number of episodes.

That is, for t = episode_count,
value(t) = initial_value * e^(-decay_rate * t)

Note

By successfully completed episodes, we mean that the episode_count only increments when the environment reaches a terminal state, not when max_steps is reached. That is, episode_count is creased by one upon receiving done=True in the context passed to the parameter.

class varcompfa.engine.parameters.EpisodicParameter(func, terminal_value=None)[source]¶

Bases: object

A parameter set according to an arbitrary single-argument function, with the argument in question being the number of successfully completed episodes.

Note

The number of episodes starts at zero, so the initial value is value(0) = func(0), where func is the provided function. By successfully completed episodes, we mean that the episode_count only increments when the environment reaches a terminal state, not when max_steps is reached. That is, episode_count is creased by one upon receiving done=True in the context passed to the parameter.

class varcompfa.engine.parameters.EpisodicPowerLaw(base, exponent, terminal_value=None)[source]¶

Bases: object

A parameter that decays according to a power law with respect to the number of completed episodes.

That is, for t = episode_count,
value(t) = base * (t+1)**(-exponent)

Note

By successfully completed episodes, we mean that the episode_count only increments when the environment reaches a terminal state, not when max_steps is reached. That is, episode_count is creased by one upon receiving done=True in the context passed to the parameter.
