varcompfa.algos.discrete_actions_q_learning module¶

Q-Learning using LFA with discrete actions.

class varcompfa.algos.discrete_actions_q_learning.DiscreteQ(num_features, num_actions, epsilon=0.05)[source]¶

Bases: varcompfa.algos.algo_base.LearningAlgorithm

Q-Learning with linear function approximation.

Actions are assumed to be discrete, while states are represented via a feature vector. That is, Q(s,a) = [〈w, x〉]_a Exploration occurs via an ε-greedy policy.
