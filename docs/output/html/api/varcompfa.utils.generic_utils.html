varcompfa.utils.generic_utils.find_module(obj)[source]¶

Find the name of the module where obj is defined.

For example, vcf.TD becomes vcf.algos.td

varcompfa.utils.generic_utils.get_class_string(obj)[source]¶

Get the class string for an object

For example, vcf.TD becomes vcf.algos.td.TD, as does an instantiation of that class.

May be slightly dicey because Python’s object model doesn’t always permit you to find the right thing. For example, a numpy array returns numpy.ndarray, which is the correct type, but attempting to instantiate it the same way you would create an array via numpy.array([1,2]) will yield a zero-valued array with shape (1,2), rather than an array of shape (2,) with values [1,2].
